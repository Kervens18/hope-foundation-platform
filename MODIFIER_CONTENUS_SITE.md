# Modifier les contenus du site via Netlify CMS

## Modifier les Projets
1. Accédez à l’admin : `/admin`
2. Cliquez sur « Produits » (ou « Projets » selon le nom affiché)
3. Cliquez sur un projet existant pour le modifier, ou sur « New Produits » pour en ajouter un nouveau
4. Modifiez le nom, la description, le prix, l’image, etc.
5. Cliquez sur « Publish » pour enregistrer

## Modifier la Galerie Photo
1. Dans l’admin, cliquez sur « Galerie Photo »
2. Cliquez sur une photo pour la modifier, ou sur « New Galerie Photo » pour en ajouter une nouvelle
3. Modifiez le titre, l’image, la description
4. Cliquez sur « Publish »

## Modifier les Actualités (Blog)
1. Dans l’admin, cliquez sur « Articles de Blog »
2. Cliquez sur un article pour le modifier, ou sur « New Articles de Blog » pour en ajouter un nouveau
3. Modifiez le titre, la date, l’auteur, l’image principale, le contenu
4. Cliquez sur « Publish »

---

**Astuce :**
- Toutes les images sont téléversées via l’admin, pas besoin d’accès technique.
- Les modifications sont visibles sur le site après rafraîchissement de la page d’accueil.
- Pour supprimer un contenu, ouvrez-le puis cliquez sur « Delete ».
