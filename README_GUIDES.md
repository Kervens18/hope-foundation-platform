## 🔥 Questions Fréquentes

### Comment modifier les projets?

**Réponse courte:** Éditez `js/config.js`, array `projects[]`

**Guide complet:** [QUICK_START_PROJECTS.md](QUICK_START_PROJECTS.md)

### Comment ajouter des images?

**Réponse courte:** Placez JPG dans `/images/` nommé `project-1.jpg`, etc.

**Guide complet:** [PROJECTS_GUIDE.md](PROJECTS_GUIDE.md#comment-ajouter-une-image)

### Comment déployer en ligne?

**Réponse courte:** GitHub Pages (gratuit) ou Netlify (facile)

**Guide complet:** [DOMAIN_CONFIG.md](DOMAIN_CONFIG.md)

### Comment ajouter les paiements?

**Réponse courte:** Intégrer Stripe ou PayPal

**Détails:** [NEXT_STEPS.md](NEXT_STEPS.md#6%EF%B8%8F⃣-configurer-les-paiements-payant)

### Comment ajouter Google Analytics?

**Réponse courte:** Ajouter script Google GA dans `index.html`

**Détails:** [NEXT_STEPS.md](NEXT_STEPS.md#7%EF%B8%8F⃣-ajouter-google-analytics)
